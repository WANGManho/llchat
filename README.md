# English Chat Tutor

Practice English with an AI conversation partner and get instant feedback on grammar, word choice, and better expressions.

## Getting Started

Set your OpenAI API key, then run the development server:

```bash
npm install
cp .env.example .env.local
```

Edit `.env.local` and add your `OPENAI_API_KEY` (optionally `OPENAI_MODEL`).

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
